Bước 1: Cài đặt công cụ đóng gói Sử dụng PyInstaller để biến code Python thành file .app:
pip install pyinstaller
Bước 2: Tạo file .app từ code Python Giả sử file chính của bạn là main.py, chạy lệnh sau:
pyinstaller --windowed --name "MyApp" --icon=your_icon.icns LichAmDuong.py
Bước 3: Kiểm tra ứng dụng File .app sẽ nằm trong thư mục dist/. Chạy thử để đảm bảo không lỗi.
Bước 4: Tạo file .dmg từ ứng dụng .app Cách 1: Sử dụng Terminal (hdiutil):
hdiutil create -volname "MyApp" -srcfolder ./dist/MyApp.app -format UDZO -ov MyApp.dmg
